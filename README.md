# Task Management System (Secure User-Based System)

## 📌 Description
This is a secure ASP.NET Core MVC application where each user can manage their own tasks independently. The system ensures that users can only access and manage their own data.

## 🚀 Features
- User Registration and Login (Authentication System)
- Secure User-Based Access (Data Isolation)
- Create, Read, Update, Delete (CRUD) operations on tasks
- Each user can manage only their own tasks

## 🛠️ Technologies Used
- ASP.NET Core MVC
- C#
- Entity Framework Core (EF Core)
- SQL Server

## 🔐 Security
- Implemented authentication for secure access
- User-specific data handling to ensure privacy and data isolation

## ▶️ How to Run the Project
1. Clone the repository
2. Open the project in Visual Studio
3. Configure the SQL Server connection string
4. Run the application
